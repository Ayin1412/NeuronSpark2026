# 智慧饭堂人流与备餐预测复现说明

本目录存放了本题（NS-2026-01）的最终预测方案复现代码。通过运行本目录下的 `main.py`，可以从原始公开数据生成最终的平台提交文件 `results.csv`。

## 1. 运行环境要求

*   **操作系统**：Windows / Linux / macOS
*   **Python版本**：`Python >= 3.12`
*   **关键依赖库**：
    *   `lightgbm>=4.0.0`
    *   `pandas>=1.5.0`
    *   `numpy>=1.21.0`

您可以使用以下命令一键安装所需依赖：
```bash
pip install -r requirements.txt
```

## 2. 数据准备

在运行复现脚本前，请将平台提供的原始公开数据集放置在**与 `main.py` 同一目录下**。
所需数据文件包括：
*   `train.csv` (训练集)
*   `test.csv` (测试集)
*   `calendar.csv` (日历辅助表)
*   `weather.csv` (天气辅助表)
*   `menu.csv` (菜单辅助表)

## 3. 运行复现

在终端中切换到 `src/` 目录，执行以下命令：

```bash
python main.py
```

### 运行说明与输出：
1. 运行该脚本时会先进行**本地验证**（以最后80天作为验证集评估 MAE、RMSE、MAPE 并输出估算分数，本地估算得分约为 `488.2`）。
2. 本地验证结束后，脚本将自动读取全量数据进行**全量训练**（训练 6000 轮 LightGBM 模型）。
3. 训练完成后，将在当前目录下输出最终预测结果文件 **`results.csv`**。
4. 运行全过程预计耗时 **2~3 分钟**（CPU 运行，无需 GPU）。
